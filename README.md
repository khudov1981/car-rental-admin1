# Админ панель для аренды автомобилей

Telegram Mini App для управления прокатом автомобилей.

## Описание

Админ панель для аренды автомобилей - это мини-приложение для Telegram, предназначенное для управления автопарком и клиентами сервиса проката автомобилей. Приложение позволяет:

- Просматривать список автомобилей
- Добавлять новые автомобили
- Редактировать информацию об автомобилях
- Управлять статусами автомобилей (доступен, арендован, на обслуживании)
- Просматривать список клиентов (в разработке)

## Особенности

- Адаптировано для работы в Telegram как Mini App
- Поддержка тем Telegram
- Адаптивный дизайн для мобильных устройств
- Интуитивный интерфейс с навигацией внизу экрана
- Динамическое отображение имени пользователя из Telegram

## Установка и запуск

1. Клонируйте репозиторий:
   ```bash
   git clone <repository-url>
   ```

2. Установите зависимости:
   ```bash
   npm install
   ```

3. Запустите приложение в режиме разработки:
   ```bash
   npm run dev
   ```

4. Для сборки production версии:
   ```bash
   npm run build
   ```

## Использование

### Навигация

Приложение имеет навигацию внизу экрана с двумя разделами:
- **Авто** - управление автопарком
- **Клиенты** - управление клиентами (в разработке)

### Управление автомобилями

На странице "Авто" вы можете:
1. Просматривать список всех автомобилей
2. Добавлять новые автомобили через кнопку "+ Добавить авто"
3. Редактировать информацию об автомобиле через кнопку "Редактировать"
4. Удалять автомобили через кнопку "Удалить"

Каждый автомобиль отображается в виде карточки с:
- Фотографией автомобиля
- Маркой и моделью
- Годом выпуска
- Цветом
- Номером автомобиля
- Стоимостью аренды в день
- Статусом (доступен, арендован, на обслуживании)

### Запуск Telegram бота

Для запуска Telegram бота необходимо:

1. Создать бота через @BotFather в Telegram и получить токен
2. Установить токен как переменную окружения:
   ```bash
   export TELEGRAM_BOT_TOKEN="ваш_токен_бота"
   ```
3. Запустить бота:
   ```bash
   npm run bot
   ```
