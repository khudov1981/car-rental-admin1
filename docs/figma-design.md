# Работа с макетом дизайна из Figma

## Варианты передачи макета из Figma

### 1. Ссылка на общий доступ (самый простой способ)
- Сделайте макет доступным по ссылке в Figma
- Установите права доступа "Anyone with the link can view"
- Отправьте ссылку на макет

### 2. Экспорт в PDF
- Экспортируйте макет в PDF через Figma:
  - File → Export → PDF
  - Или через плагины экспорта
- Загрузите PDF файл в репозиторий или отправьте

### 3. Экспорт отдельных экранов
- Экспортируйте отдельные экраны как PNG/SVG:
  - Выберите фрейм → Export → PNG/SVG
  - Загрузите изображения в репозиторий

### 4. Использование Figma API
- Предоставьте доступ к файлу Figma
- Получение информации через Figma API

## Что необходимо для верстки

### Визуальные элементы
- Цветовая палитра (HEX/RGB коды)
- Шрифты и их размеры
- Отступы и размеры элементов

### Интерактивные элементы
- Состояния кнопок (normal, hover, active, disabled)
- Анимации и переходы
- Поведение при наведении

### Адаптивность
- Разные размеры экранов (мобильный, планшет, десктоп)
- Поведение элементов при изменении размера

## Рекомендуемый подход

### 1. Общий доступ к макету в Figma
- Установите права доступа "Anyone with the link can view"
- Отправьте ссылку команде разработки

### 2. Добавьте комментарии с пояснениями
- Пояснения к неочевидным элементам
- Описание поведения интерактивных компонентов
- Примечания по анимациям и переходам

### 3. Укажите breakpoints для адаптивной верстки
- Мобильные устройства: до 768px
- Планшеты: 768px - 1024px
- Десктопы: от 1024px

### 4. Опишите поведение интерактивных элементов
- Состояния кнопок и форм
- Поведение при наведении и фокусе
- Обработка ошибок в формах

## Пример структуры комментариев

```
Цвета:
- Основной синий: #229ED9
- Фон: #F5F5F5
- Текст: #333333

Шрифты:
- Заголовки: Roboto Bold 24px
- Основной текст: Roboto Regular 16px
- Подписи: Roboto Light 14px

Отступы:
- Между блоками: 24px
- Внутри карточек: 16px
- Поля ввода: 12px
```

## Полезные плагины Figma для экспорта

### 1. Zeplin
- Автоматический экспорт спецификаций
- Генерация CSS/SCSS кода
- Измерения и расстояния между элементами

### 2. Avocode
- Экспорт стилей и размеров
- Генерация кода для разных фреймворков
- Комментарии и аннотации

### 3. Figma to Code
- Прямая генерация HTML/CSS кода
- Экспорт компонентов как React кода
- Сохранение структуры и иерархии

## Рекомендации по подготовке макета

### 1. Используйте компоненты
- Создавайте повторяющиеся элементы как компоненты
- Используйте варианты компонентов для разных состояний
- Поддерживайте консистентность в дизайне

### 2. Организуйте слои
- Используйте понятные названия для слоев
- Группируйте связанные элементы
- Следите за порядком слоев

### 3. Добавьте автолейауты
- Используйте Auto Layout для гибких компонентов
- Задавайте правильные отступы и выравнивания
- Тестируйте адаптивность компонентов

### 4. Документируйте стили
- Создайте страницу с текстовыми стилями
- Документируйте цветовую палитру
- Опишите поведение интерактивных элементов

## Пример структуры дизайн-системы

### Цвета
- Primary: #229ED9 (кнопки, ссылки)
- Secondary: #6C757D (второстепенные элементы)
- Success: #28A745 (успешные действия)
- Warning: #FFC107 (предупреждения)
- Danger: #DC3545 (ошибки, удаление)

### Типографика
- H1: 32px, Bold, #333333
- H2: 24px, Bold, #333333
- Body: 16px, Regular, #333333
- Caption: 14px, Light, #666666

### Отступы
- XS: 4px
- S: 8px
- M: 16px
- L: 24px
- XL: 32px

## Следующие шаги

1. Подготовьте макет в Figma
2. Сделайте его доступным по ссылке
3. Добавьте необходимые комментарии и пояснения
4. Отправьте ссылку команде разработки
